# Cognitive Science Week 3 Notes

## Topic
Memory systems and retrieval practice

## Key Points
- Working memory is capacity-limited
- Long-term retention improves with retrieval practice
- Spaced repetition outperforms cramming
- Interleaving can strengthen discrimination between similar concepts
